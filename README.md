# AI Audiobook Generator 🎧

## Features
- Upload PDF/DOC/TXT
- Extract text
- Rewrite using AI
- Convert to speech
- Generate audiobook

## Tech Stack
- Python
- Streamlit
- NLP
- Text-to-Speech

## ⚠️ Environment Setup

### Main App (Python 3.13)
pip install -r requirements.txt

### SadTalker (Python 3.10)
Setup separately due to compatibility

## ▶️ Run App
streamlit run app.py
